### 핵심 요약

### 💡 왜 중요한가

이 기사는 빅테크 기업의 실적 평가 방식이 변화하고 있음을 보여주며, 개발자와 IT 종사자들은 이러한 변화가 기업 전략과 투자 결정에 미치는 영향을 이해해야 한다. 새로운 기준은 기술 투자와 자본 지출의 중요성을 강조하고 있다.

## 14. GitHub 장애 - Incomplete pull request results in repositories Incident Report for GitHub

🔗 [원문 보기](https://www.githubstatus.com/incidents/x69zbgdyfzg0)

### 핵심 요약

- GitHub에서 Pull Request(PR) 검색 기능에 장애가 발생했다.
- 장애 원인은 ElasticSearch 인덱스 재구축 중 발생한 문제이다.
- PR 데이터 자체는 손실되지 않았으나 검색 기능이 제한되고 있다.
- GitHub 측은 약 24시간 내에 완전 복구를 예상하고 있다.
